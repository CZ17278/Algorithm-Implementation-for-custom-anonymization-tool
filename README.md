# UNITBV Adaptive Anonymization Tool (UAAT)

## Overview

The **UNITBV Adaptive Anonymization Tool (UAAT)** is a Python-based system designed to anonymize datasets while preserving data utility. It addresses the limitations of existing tools such as Amnesia (high information loss) and ARX (complex configuration), by providing an **automated, user-friendly, and adaptive anonymization process**.

The system supports:
- k-anonymity
- l-diversity
- t-closeness

---

## Key Features

- Automated generalization (no expert tuning required)
- Iterative anonymization strategy (levels 0–3)
- Integrated privacy validation
- Risk and utility evaluation
- Graphical User Interface (GUI)
- Designed for datasets up to 10,000 records
